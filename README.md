# Dotfiles

## How to install?

1. Clone repository
```sh
git clone git@github.com:ysomad/dotfiles.git
```

2. Go to dotfiles directory
```sh
cd /path/to/dotfiles
```

3. Run
```sh
./bootstrap.sh
```
It will install Homebrew, oh-my-zsh, link configs and install brew dependancies along side with macOS sensible defaults
