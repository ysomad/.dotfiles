# Dotfiles

## How to install?

1. Clone repository
```sh
git clone git@github.com:ysomad/dotfiles.git
```

2. Go to dotfiles directory
```sh
cd /path/to/dotfiles
```

3. Make bootstrap script executable
```sh
chmod +x ./bootstrap.sh
```

4. Run
```sh
./bootstrap.sh
```
